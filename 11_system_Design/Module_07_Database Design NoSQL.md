# 81. When to Choose NoSQL?

## Definition

NoSQL (Not Only SQL) databases are non-relational databases designed for:

- High scalability
- Flexible schemas
- Large distributed workloads
- High throughput

Examples:

- MongoDB
- Cassandra
- DynamoDB
- Redis
- Neo4j
